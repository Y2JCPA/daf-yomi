Minimum Garment Size for Tzitzit

A garment is obligated in tzitzit only if it's large enough for a minor to cover himself (head and most of the body).

If an adult occasionally goes out wearing it → obligated.
If it's too small even for a minor → exempt.

Can You Transfer Tzitzit Between Garments?

Baraita: One may untie tzitzit from one garment and re-tie them on another — provided the strings are not broken.

This seems to conflict with "ta'aseh v'lo min ha'asui" (must be made fresh). The resolution: since you're actively re-tying them onto the new garment, that IS an act of "making."

Does the String Color Need to Match the Garment?

The Rabbis

  • Any color string exempts any color garment
  • White strings on a blue garment? Fine!
  • Source: "with which you cover yourself" — focuses on the GARMENT, not string color

Dissenting View

  • Strings should match the garment color
  • Blue garment → blue strings
  • Red garment → red strings
  • Only the tekhelet string is always blue regardless

A Cloak Entirely of Sky-Blue Wool

Baraita: A cloak made entirely of sky-blue wool — strings of every color exempt it. You don't need sky-blue strings just because the garment is blue.

But: a garment is exempted only by strings of its own type. How to reconcile? The "own type" means material (wool for wool, linen for linen), not color.

Four Corners — Why Not Three or Five?

The verse says "corners" (כנפות) — how many?

• 4 corners: obligated ✅
• 5+ corners: obligated ✅ (put tzitzit on 4 of them)
• 3 corners: exempt ❌

Why include 5 but exclude 3? "With which you cover yourself" → include a garment with more corners (still wearable). A 3-cornered garment is not a typical covering.
7 / 7

Key Takeaways — Menachot 40

1. Minimum size: large enough for a minor to cover head and body.

2. May transfer tzitzit between garments if strings are intact.

3. Any color string works on any color garment (Rabbis' view).

4. "Own type" = material (wool/linen), not color.

5. 4+ corners obligated; 3 corners exempt.
