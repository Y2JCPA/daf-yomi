How to Test Genuine Tekhelet

Test recipe (40-day old snail solution):
1. Soak the sky-blue wool in the snail extract from night until morning
2. If the color fades → invalid (plant dye, not chilazon)
3. If the color holds → valid tekhelet

Rav Adda / Rav Avira's alternative test: Bake the wool in leavened barley dough. Color holds → genuine. Color fades → counterfeit.

But Wait — Is There a Reliable Test?

Baraita: "There is no reliable method of testing sky-blue wool."

This doesn't mean the tests above don't work — it means there's no way to tell if the dye source was genuine chilazon or a plant that produces an identical, equally permanent color (kala ilan).

Bottom line: you need to trust the seller. Buy only from a reputable, established dealer.

"With Which You Cover Yourself" (Deut. 22:12)

The verse "on the four corners of your garment with which you cover yourself":

The Rabbis use "with which you cover yourself" to exclude a night garment — tzitzit is a daytime obligation only (since you "cover yourself" for going out, not sleeping).

But wait — the accepted halakha is that night garments ARE obligated. So the Rabbis use this verse for a different teaching.

How Many Corners?

CornersObligated?Reasoning
3❌ Exempt"Four" = minimum four
4✅ ObligatedExplicit in the verse
5+✅ Obligated"Your garment" includes garments with more corners
Why include 5 but exclude 3? A 4-cornered garment is the prototype. 5+ corners includes the 4; 3 corners doesn't reach the minimum.

Three Blessings of Distinction

R. Meir: A person must recite three blessings daily:

1. "...who did not make me a gentile"
2. "...who did not make me a woman"
3. "...who did not make me an ignoramus / slave"

These are blessings of gratitude for the obligation to perform mitzvot — not statements of superiority.

Key Takeaways — Menachot 42

1. Test tekhelet: soak in snail extract or bake in barley dough. Color must hold.

2. No definitive test for dye SOURCE — must trust the seller.

3. "With which you cover yourself" — debated whether it excludes night garments.

4. 4+ corners: obligated. 3: exempt.

5. Three daily blessings of distinction — gratitude for mitzva obligations.
