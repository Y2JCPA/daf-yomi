What Counts as 'Vegetables'? What Counts as 'Meat'?

Mishna
One who vows vegetables are forbidden — may he eat gourds?
Rabbis
Gourds are permitted — people don't call gourds 'vegetables'
R' Akiva
Gourds are forbidden — an agent who can't find vegetables brings gourds, not legumes
Ulla
The case is one who said: 'Vegetables cooked in a pot' are forbidden — gourds fit that category
Core Question When someone makes a vow using a general term, how broadly do we interpret it? The Gemara uses the agent consultation test to determine if sub-categories are included.

Gourds as Vegetables: The Agent Argument

Rabbis
A person tells his agent: Buy me vegetables. The agent returns and says: 'I found only gourds.' This proves gourds are not vegetables!
R' Akiva
Exactly my point! The agent says 'I found only gourds' — but would he ever say 'I found only legumes'? No! Gourds are a sub-type of vegetables; legumes are not.
Rabbis
The agent consults before buying gourds — that shows they're a different category
R' Akiva
Consulting proves it's the same type — for a truly different type, the agent wouldn't even ask!
Key Principle R' Akiva: If an agent consults before substituting, it's the same category. If he wouldn't even consider substituting, it's a different category entirely.

The Agent Consultation Test

Agent is sent to buy Item X
↓
Agent cannot find Item X. Does he consider buying Item Y as a substitute?
Yes — Consults employer
Rabbis: Different type (permitted)
R' Akiva: Same type (forbidden)
No — Wouldn't even consider it
Both agree: Completely different type (permitted)
Abaye's Qualification Even R' Akiva concedes: one who ate gourds after vowing off vegetables is not flogged, since the matter is not entirely clear.

Ulla's Clarification: What Did He Actually Vow?

Problem

  • Gourds are technically fruits, not vegetables
  • How can R' Akiva include them in a vegetable vow?
  • The vow must be interpreted more specifically

Ulla's Solution

  • He vowed: 'A vegetable cooked in a pot is forbidden'
  • Gourds ARE cooked in pots alongside vegetables
  • Not 'vegetable that flavors the pot' — but 'vegetable cooked in a pot'
Precision The Gemara distinguishes between ירק שבקדירה (vegetable cooked in a pot) and ירק הנאכל בקדירה (vegetable eaten as pot flavoring) — the exact wording determines whether gourds are included.

R' Shimon b. Gamliel's Saying Innards are not meat, and one who eats them is not a person — meaning: for vows, innards count as meat; for purchase, buying innards doesn't qualify as buying meat.

Me'ila: Agent Deviation & Consecrated Property

📋 Employer sends agent to use a specific item (forgetting it's consecrated — hekdesh)
↓
✅ Agent performs mission properly → Employer is liable for me'ila
↓
❌ Agent deviates from mission → Agent is liable for me'ila
↓
🔍 Example: Employer says 'Give meat,' agent gives liver → Agent deviated = agent liable
↓
⚖️ Rav Ḥisda: This cannot follow R' Akiva — he'd say liver IS meat, so agent fulfilled his mission!
↓
💡 Abaye: Even R' Akiva agrees — the agent should have consulted first; failing to consult = deviation
Rava's Verdict Rava affirmed Abaye's answer: Naḥmani spoke well. Even under R' Akiva's view, the agent's failure to consult constitutes deviation from his mission.
