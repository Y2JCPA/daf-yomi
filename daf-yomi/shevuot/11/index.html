The Nature of Sanctity in Surplus Public Offerings

Rav Ḥisda
Surplus incense has only kedushat damim (sanctity of value) — so how does it lose that sanctity when sold?
Rabba
If it has only value-sanctity, it should NOT be disqualified by a tevul yom — but a baraita says it IS disqualified once placed in the mortar!
Rav Ḥisda
If you say it has inherent sanctity, it should be disqualified by being left overnight — but the Mishna shows it is NOT!
Rabba
Incense is different — its form doesn't change, so linah (overnight disqualification) doesn't apply to it.
Rabba
The court tacitly stipulates: if needed this year → inherent sanctity; if not needed → value-sanctity only, so it can be sold.
Core Issue Can surplus communal offerings lose their sanctity? Rabba proposes the court builds in a conditional stipulation at the time of consecration.

Two Types of Sanctity (Kedusha)

Kedushat HaGuf

  • Inherent sanctity in the item itself
  • Cannot be removed without blemish
  • Disqualified by tevul yom
  • Disqualified by leaving overnight (usually)
  • Example: Animal offerings on the altar

Kedushat Damim

  • Sanctity inheres only in the item's value
  • Can be redeemed/sold
  • NOT disqualified by tevul yom
  • NOT disqualified by leaving overnight
  • Example: Items verbally dedicated before placed in service vessel
Key Distinction Meal-offering ingredients gain inherent sanctity only when placed in a kli sharet (service vessel). Before that, they have only value-sanctity.

Rabba vs. Rav Ḥisda: The Sanctity of Incense

Rav Ḥisda
Surplus incense has only kedushat damim. My question stands: where did its sanctity go when sold?
Rabba
A baraita teaches: once placed in the mortar, incense is disqualified by a tevul yom. Only items with inherent sanctity are disqualified that way!
Rabba
Proof from Mishna Me'ila 9a: meal-offering ingredients with only value-sanctity are NOT disqualified by tevul yom — only after being placed in a service vessel.
Rav Ḥisda
Counter-proof from Mishna Me'ila 10a: incense is disqualified by linah (overnight) only after being placed in a service vessel. A mortar isn't a vessel — so it still has only value-sanctity!
Rabba
Incense is different regarding linah — its form never changes throughout the year. Overnight disqualification only applies to substances that spoil!
Stalemate Rav Ḥisda's original question remains: if incense has inherent sanctity, how does the court remove it when selling the surplus?

Rabba's Solution: The Court's Tacit Stipulation (Tena'i Beit Din)

Court consecrates items for public offerings (daily offerings, incense)
↓
Built-in tacit condition at time of consecration
If needed this year
Full kedushat haguf — inherent sanctity as offerings
If NOT needed
Only kedushat damim — value-sanctity, so items can be sold
Resolution This explains how surplus incense can be sold to the same priestly watch: the stipulation retroactively limits its sanctity to value only.
6 / 9

Abaye Challenges the Stipulation Theory

Abaye
Didn't you, Rabba, teach that if one consecrates a ram for its value, since the ram itself is fit as an offering, it automatically receives inherent sanctity? The stipulation should be useless!
Rabba
That applies only when one says 'for the value of a burnt-offering' (for which a ram is suitable). The court's stipulation is like saying 'for the value of libations' — the item isn't suitable for that, so no automatic upgrade.
Abaye
Lost bull and goat of Yom Kippur: if found after replacements were offered, they must graze until blemished. Why not use the stipulation to simply redeem them?
Rabba
Lost animals are uncommon — the court doesn't stipulate for uncommon events.
Abaye
But the red heifer is also uncommon, yet a baraita teaches it CAN be redeemed before slaughter — implying a stipulation!
Rabba
The red heifer is very expensive. To prevent great loss, the court does stipulate even for uncommon cases.
Principle The court's stipulation applies to: (1) common cases like surplus offerings, and (2) uncommon cases involving great financial loss, like the red heifer.
