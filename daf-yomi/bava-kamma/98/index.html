Bava Kamma 98: Rabba's Exemptions for Indirect Damage

דבר הגורם לממון — כממון דמי?
KEY THEME Rabba establishes a series of rulings exempting one from liability when damage is indirect or non-substantial: throwing a coin into clear water, defacing a coin's image, slitting an animal's ear, and burning a promissory note. Rava challenges each ruling from baraitot, and the Gemara resolves each objection. The daf concludes with the concept of davar ha-gorem l'mamon — whether an item that merely causes financial loss has monetary value.

Defacing a Coin's Image — Hammer vs. File

Rabba
One who effaces the image on another's coin is exempt. He did not reduce its size — the coin remains intact.
Gemara
This applies only if he struck it with a hammer (kurnesa) and flattened it.
Gemara
But if he filed it (shofina), he must pay — because the coin diminished in size.
Rava
But a master who strikes a slave's eye and blinds it — the slave goes free! Invisible damage still counts!
Rabba
Deafening causes a physical change — a drop of blood falls inside the ear. A hammered coin has no internal change.
KEY DISTINCTION Damage must be substantial — a coin struck flat still has its full metal content, while filing physically removes material.

Rabba's Four Exemptions from Liability

Coin in the Sea

  • Translucent water only
  • "Your coin is before you"
  • Rolled, not picked up
  • Item still accessible

Defaced Coin

  • Hammer only, not file
  • No reduction in size
  • Image gone but metal intact
  • Non-substantial change

Slit Cow's Ear

  • Cow still usable for all prior uses
  • Disqualified as offering only
  • "Not all oxen stand near the altar"
  • Theoretical loss, not practical

Burned Note

  • "I burned only your paper"
  • Debt value not in the paper itself
  • Witnesses can rewrite the note
  • Indirect financial damage
COMMON THREAD Rabba exempts damage that is indirect, non-substantial, or where the essential value remains accessible.

Davar HaGorem L'Mamon — Does It Have Value?

R' Shimon
An item that causes financial loss (davar ha-gorem l'mamon) is considered to have monetary value. One who burns a promissory note is liable.
Rabbis
An item that merely causes financial loss is not considered to have monetary value. Burning a note — exempt.
Rabba
I hold like the Rabbis. The paper has no inherent debt-value.
Rav Huna b. Rav Yehoshua
R' Shimon's rule may apply only to items with intrinsic value (like chametz) — not to a promissory note with no inherent worth.
Ameimar
If we hold garmi (indirect causation) is liable — pay full debt value. If not — pay only the value of the paper.
Rafram
I forced Rav Ashi to pay damages for a document he burned in his youth — the full debt value.
HALACHA Rafram's ruling against Rav Ashi shows the practical acceptance of dina d'garmi — liability for indirect causation of financial loss.
