⚖️ Why Tereifa's Disqualification Requires a Verse

Goal: Prove a tereifa is unfit for sacrifice without a verse
↓
Attempt 1: A fortiori from blemished animal
Blemished animal is unfit → tereifa should be too
Refutation
Unique to blemished: Torah equated blemished priests with blemished animals — no parallel for tereifa
↓
Attempt 2: Common element (blemished + caesarean)
Both unfit for sacrifice → tereifa fits the pattern
Refutation
Their common element: never permitted for sacrifice. But tereifa was 'permitted' via bird sin-offering pinching!
↓
Conclusion (Rav Ashi): A verse is essential — all logical paths are blocked by unique features

⚔️ Rav Ashi vs. Rav Aḥa Sava: Can Logic Disqualify a Tereifa?

Rav Ashi
The a fortiori fails from the start. A blemished animal has a unique feature: the Torah equated blemished priests with blemished animals. No such parallel exists for tereifa.
Rav Aḥa Sava
But a caesarean-born animal proves otherwise — the Torah did NOT equate priests with it, yet it is still unfit! The 'blemished priest' feature therefore does not block the inference.
Rav Ashi
A caesarean-born has its own unique feature: it cannot receive firstborn sanctity, whereas a tereifa firstborn IS sanctified. This blocks using it as proof.
Rav Aḥa Sava
Combine both! Their common element is: permitted to humans, forbidden to the Most High. Neither was ever permitted for sacrifice — so tereifa should be the same.
Rav Ashi
That common element has its own unique feature: their prohibition was never lifted. But tereifa was effectively 'permitted' — the pinching of a bird sin-offering works even on a tereifa bird. The verse remains necessary!
Bottom Line Every logical path to disqualifying a tereifa is blocked — Scripture alone suffices

🌾 Mishna: Actions and Persons That Disqualify Handful Removal

Non-priest (זר) removed the handful → offering unfit
↓
Acute mourner (אונן) — relative died, not yet buried → unfit
↓
Immersed that day (טבול יום) — awaiting sunset after immersion → unfit
↓
Lacking atonement (מחוסר כיפורים) — awaiting final offerings → unfit
↓
Impure priest who served while impure → unfit
↓
Left hand used for removal → Ben Beteira: return handful to vessel, redo with right hand
↓
Stone, salt, or frankincense emerged with the handful → unfit (handful is 'outsized' or 'lacking')

🔍 Three Unfit Animals — Key Distinctions

Blemished Animal (בעל מום)

  • Unfit for sacrifice
  • Permitted for human consumption
  • Receives firstborn sanctity
  • Torah equated blemished priests with it
  • Prohibition never temporarily lifted for the altar

Caesarean-Born (יוצא דופן)

  • Unfit for sacrifice
  • Permitted for human consumption
  • Does NOT receive firstborn sanctity
  • Torah did NOT equate priests with it
  • Prohibition never temporarily lifted for the altar

Tereifa (טריפה)

  • Unfit for sacrifice — needs a verse
  • Permitted for human consumption
  • A tereifa firstborn IS sanctified
  • No priestly parallel
  • Prohibition was 'lifted': bird sin-offering pinching is valid even on a tereifa

🔁 Rav Naḥman's Resolution: When Is the Handful Rite Complete?

Unfit person removes the handful (non-priest, left hand, etc.)
↓
Is this removal a complete sacrificial rite?
If fully complete at removal
Disqualifies immediately → return should be impossible.
But then why does ben Beteira ever allow return?
If not a rite at all
Should always be returnable and never disqualify.
But the mishna says it does disqualify!
↓
Rav Naḥman's Answer: Removal begins the rite — but it is only complete when the handful is placed in a service vessel. Before placement, ben Beteira permits return and a valid redo.
