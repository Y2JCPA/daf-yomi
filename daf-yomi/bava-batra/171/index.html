⚖️ The Great Receipt Dispute

Rabbi Yehuda
When a debtor repays part of his debt, the original promissory note is torn up and a new document is written for the remaining balance, dated from the original date.
Rabbi Yosei
We write a receipt for the debtor while keeping the original promissory note intact. Two reasons: (1) psychological coercive pressure, and (2) the creditor should not lose his claim if the debtor later loses the note.
Rabbi Yehuda
A receipt is dangerous! A postdated note combined with a lost receipt could allow a creditor to collect twice. Better to tear the note and write fresh.
Rabbi Yosei
If there is no receipt and the creditor loses the note, should the debtor simply eat and rejoice? The receipt protects both parties equitably.
Gemara's Conclusion The halakha follows Rabbi Yosei — one writes a receipt. As Ravin reported from Eretz Yisrael in the name of Rabbi Ilai: 'The halakha is that one writes a receipt.' Rav Safra then solved the double-collection risk by instructing courts to write 'paid' on the original note.

📜 Two Approaches to Partial Repayment

Rabbi Yehuda

  • Tear the original promissory note
  • Write a new document for the remaining balance
  • New note dated from the original date — preserves lien on old property
  • If creditor loses note → debtor need not repay
  • No receipts = no double-collection risk from postdated notes

Rabbi Yosei

  • Keep the original promissory note intact
  • Write a receipt (שובר) for the amount paid
  • Creditor retains coercive power over debtor
  • If creditor loses note → debtor still owes (receipt proves partial payment only)
  • ⚠️ Risk: postdated note + lost receipt = possible double collection

🏠 Postdated Bills of Sale — Rav Hamnuna's Ruling

Rav Hamnuna
The Mishnah says postdated promissory notes are valid. But postdated bills of sale are invalid!
Gemara
Why the distinction? What harm could a postdated bill of sale cause that a promissory note cannot?
Rav Hamnuna
If Reuven sells a field to Shimon on 1 Nisan but dates the deed 1 Tishrei (6 months earlier), then Reuven's creditor can seize the field from Shimon — the field appeared to be Reuven's during those 6 months!
Gemara
Couldn't the same harm arise from postdated promissory notes? A false date makes it appear the borrower had assets earlier, enabling seizure of property bought in between...
Rav Hamnuna
I hold like Rabbi Yehuda: one does not write a receipt. A lost promissory note means no repayment obligation — so the double-collection scenario never arises for loans.
Key Distinction Bills of sale: postdating harms third-party buyers who acquired land between the real and written dates. Promissory notes (per Rabbi Yehuda): no receipts are written, so postdating cannot enable double collection.

🔧 Resolving the Double-Collection Problem

The Problem: Rabbi Yosei says write a receipt. But if the receipt is lost, the creditor can collect again using the original note — double collection!
↓
Rav Huna b. Rav Yehoshua's Attempt: Write a receipt only for partial repayment — not for full repayment. This limits the window of risk.
↓
Gemara Rejects This: The case of Rav Yitzchak bar Yosef vs. Rabbi Abba, adjudicated by Rabbi Ḥanina bar Pappi, proves we write a receipt even for full repayment of a debt.
↓
Abaye's Counter-Argument: If no receipt → a debtor whose creditor lost the note 'eats and rejoices.' If we do write receipts → a creditor whose receipt is lost suffers equally. Both systems have symmetric risks.
↓
Rav Safra's Solution: Courts write directly on the promissory note: 'This note has been paid (פרוע).' The original document becomes unenforceable — double collection is impossible.
